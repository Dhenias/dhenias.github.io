Kejuruan Animasi - SMKN 2 Cikarang Barat

 Nama saya Dhenias Syafe'i Suryawijaya, siswa jurusan Animasi di SMKN 2 Cikarang Barat semester ke-4, berada di divisi Animasi 2D. Sejak duduk di bangku SMP, saya telah memiliki ketertarikan pada animasi, khususnya animasi 2D khas Jepang (anime). Saya terbiasa menggunakan perangkat lunak Toon Boom Harmony dan Clip Studio Paint dalam pembuatan animasi. Saya juga aktif mengikuti berbagai proyek kolaborasi yang membentuk saya menjadi pribadi yang disiplin, bertanggung jawab, dan mudah beradaptasi.

 Saya senang mengeksplorasi teknik baru, mengembangkan karakter, serta mengikuti perkembangan tren industri kreatif. Bagi saya, animasi merupakan media untuk bercerita dan memberikan inspirasi. Saya berkomitmen untuk terus berkembang dan berkontribusi di industri animasi Indonesia, serta bercita-cita menjadi animator profesional.
